# 再テスト項目一覧

## P0: 主要機能の回帰確認

### 1. レポート PDF 保存
- レポートを開く。
- 「💾 PDF保存」を押す。
- JavaScript エラーが出ないこと。
- PDF ファイルがダウンロードされること。

### 2. 解析動画保存
- 動画ファイルを読み込む。
- 解析を最後まで実行する。
- 「💾 解析動画」を押す。
- 保存された動画が静止画ではなく、元動画の再生に合わせて解析オーバーレイ付きで出力されること。
- 保存後も通常の動画再生・再解析が継続して行えること。

### 3. セッション保存エラー時の通知
- IndexedDB 保存を失敗させる条件で解析終了させる。
- 「保存しました」と誤表示されず、エラー通知のみ表示されること。

### 4. MediaPipe 初期化失敗時の導線
- ネットワーク遮断や CDN ブロックで MediaPipe 初期化を失敗させる。
- 初期化完了通知が表示されないこと。
- 初期化失敗が UI 上で明確に分かること。

## P1: 画面/UI の整合性確認

### 5. レポート内チャート
- レポートを開く。
- 膝・股関節チャートに加え、時系列チャートが表示されること。

### 6. 解析動画ボタンのスタイル
- メイン画面で「💾 解析動画」ボタンの色・ホバーが他ボタン同様に適用されていること。

### 7. 画面向き警告
- モバイルまたはモバイルエミュレーションで確認する。
- 前額面で横向きにした際に警告が表示されること。
- 矢状面で縦向きにした際に警告が表示されること。
- 推奨向きに戻すと警告が消えること。

## P2: 既存主要機能の回帰確認

### 8. カメラ解析
- カメラ起動後にプレビューが継続表示されること。
- 録画開始/停止ができること。
- 録画保存が正常に行えること。

### 9. 動画解析
- 動画読み込み後に先頭フレームが表示されること。
- 「▶ 開始」で解析が進み、終了後に CSV / レポート / 解析動画保存が有効になること。

### 10. エクスポート系
- CSV エクスポートが成功すること。
- JSON エクスポートが成功すること。
- 印刷プレビューが開けること。
